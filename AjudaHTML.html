2. Dicionário de Tags e Siglas
I/A - Internação e Admissão

Indica o regime de entrada do paciente no hospital:

ANTE Internação Antecipada: 02 dias antes da cirurgia.
Vesp Véspera: Interna no dia anterior à cirurgia.
Ambu Ambulatorial: Alta prevista para o mesmo dia.
Dia Dia: Interna no mesmo dia da cirurgia.
Inter Internado: Paciente já se encontra no leito.
Clin Clínico: Internação clínica.

1. Barra Lateral Colorida (Status Geral)
Cinza: Agendado (Estado padrão).

2. Status do Guichê (Bolinha)

Localizado no topo do card. Serve para controle de recepção (Check-in).

Vazio: Nenhuma verificação feita.
OK: Check-in completo e sem pendências.
Atenção: Pendência de documentos ou exames.
Erro: Impedimento crítico na recepção.
3. Autorização (Círculo AUT)

Indica a situação junto ao convênio. Duplo clique para alterar.

4. Visão Detalhada (Modal)

Ao clicar no botão VER DETALHES, você acessa a ficha completa.

Área de Recepção (Acordeão Amarelo)

Esta área é editável e serve para anotações rápidas do guichê.

  • Check 1 (Email/AA): Campo para registrar email ou dados de contato.
  • Check 2 (Obs/AB): Observações gerais de atendimento.
Informações Técnicas
  • Dados do Paciente: Telefone, Matrícula (copiável), Nascimento.
  • Info Técnica (Editável): Campo especial para a enfermagem/anestesia inserir observações sobre materiais, alergias ou necessidades de sala. Clique no texto para editar.
  • Recursos: Lista completa de Instrumentais, Equipamentos e Exames solicitados no formulário.
5. Ações (Agendar e Mover)

➕ Novo Agendamento:
Use o botão azul no topo direito. O formulário pedirá Sala, Horário e Duração. O sistema valida automaticamente se o horário está livre.

🔄 Reagendar (Botão Amarelo):
Para trocar um paciente de dia, sala ou horário:

  1. Clique no ícone no rodapé do card.
  2. Selecione a Data de Destino.
  3. Selecione a nova Sala (o sistema só mostra as que têm vaga).
  4. Defina o novo Horário.
  5. Confirme. O paciente será removido do dia atual e criado no novo dia.

* Atenção: Ao reagendar, o status original do paciente pode ser alterado para "Remarcado".
